# thermometer
vue+zrender体温单
## 项目构成
项目现在分为花都版本(huaDu)和聊城版本(liaoCheng)。

2021.7.15 新增贵州版本(guiZhou)

项目主要代码拆分在projects文件夹里，根据命令行传入参数来区分项目。

项目需求文档后续统一放到docs文件夹里。
